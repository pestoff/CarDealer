# Car Dealer

 This is kotlin learning application. It contains 3 screens: list of the cars with add buttons, car add-on screen and car details screen. User can add and delete (by swipe) car from the list. 

## Technologies
 - Kotlin
 - MVVM
 - Fragments
 - LiveData
 - Coroutines
 - Room
 - Glide 
 - Intents
